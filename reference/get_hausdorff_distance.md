# Hausdorff distance between two streamlines

Computes the symmetric Hausdorff distance between two
[streamline](https://astamm.github.io/fiber/reference/streamline.md)
objects based on their 3-D coordinate matrices.

## Usage

``` r
get_hausdorff_distance(x, y)
```

## Arguments

- x:

  A [streamline](https://astamm.github.io/fiber/reference/streamline.md)
  object.

- y:

  A [streamline](https://astamm.github.io/fiber/reference/streamline.md)
  object.

## Value

A non-negative numeric scalar.
